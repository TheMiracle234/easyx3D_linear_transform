# easyx3D_linear_transform_tmptmp
Use easyx to visualize the 3d graph with linear transformation functions.

IDE：visual studio

Dependency：easyx.h

Change the character set to Multi-Byte Character Set in Properties → Advanced → Character Set.

operation：

wsad: ⬆️⬇️⬅️➡️

shift,space: down,up

zxc: rotation transformations around x,y,z axes

bnm: opposite directions of zxc

Numeric Keypad (General Linear Transformations):

1/2/3: Transform i-vector

4/5/6: Transform j-vector

7/8/9: Transform k-vectorp：改变线性变换的方向

p:change the direction of general linear transformations


q: show the matrix of linear transformations

e：show structure lines of how graphs are made

r: restore transformations

esc：exit
